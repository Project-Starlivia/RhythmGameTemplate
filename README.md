# RhythmGameTemplate
BMSを用いたUnityでの音楽ゲーム、リズムゲームを可能な限りシンプルに実装しています。

書典19で頒布するIwakenLab Techbook vol.7のへのへのんの章のサンプルプロジェクトのリポジトリ。
